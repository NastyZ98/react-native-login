# react-native-login
Simple login app using React Native, React Native Elements, Firebase, Redux and Redux-Thunk

## Dependencies
 * react-redux
 * react-native-elements
 * react-native-vector-icons
 * redux
 * redux-thunk
 * firebase
```
npm i
```

## Launch on emulator
Android
```
react-native run-android
```
IOS
```
react-native run-ios
```
